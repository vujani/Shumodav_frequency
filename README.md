# Shumodav_frequency
Приложение к работе по шумоподавлению в частотной области

1) Отдельное обучение - здесь модели обучаются по одному типу шума 10,15,30 и т.д
2) Смешанный вариант - здесь модели обучаются по смешанному типу шума, в обучающией выборке могут быть разные уровни шумов

Обе модели обучаются на датасете DIV2K, и должны находится в той же папке что и файлы и имет следующую структуру DIV2K/Train
